# 경험치 측정 기능 구현 순서

> 구현 시 체크리스트. 1→2→3→4→5→6 순서로 진행.

---

## 1. 화면 공유 버튼

- [x] "화면 공유 시작" 버튼 배치
- [x] `getDisplayMedia({ video: true })` 로 게임 화면 스트림 획득
- [x] 스트림을 state에 저장 (나중에 video/canvas에 연결)

---

## 2. 공유된 화면 표시

- [x] 받은 스트림을 화면에 표시 (video 또는 canvas에 그리기)
- [x] 사용자가 게임 화면이 보이도록 배치

---

## 3. 영역 선택 (사각형)

- [x] 공유 화면 위에서 드래그로 사각형 영역 지정
- [x] 선택한 영역 좌표(x, y, width, height) 저장
- [x] 선택 영역을 시각적으로 표시 (테두리 등)

---

## 4. OCR로 경험치 읽기

- [x] 선택된 사각형 구역만 캔버스에서 잘라서 이미지 생성
- [x] Tesseract.js 로 해당 이미지 OCR 실행
- [x] 인식된 문자열에서 숫자만 파싱 → 현재 경험치로 사용

---

## 5. 현재 경험치 표시 + OK → START

- [x] OCR로 읽은 "현재 경험치"를 화면에 표시
- [x] 사용자가 OK 버튼 클릭 → 이 시점 경험치 확정
- [x] OK 후 "START" 버튼 노출
- [x] START 클릭 시 측정 시작 (시작 경험치·시작 시각 기록)

---

## 6. START 후 실시간 표시

- [x] 주기적으로(예: 1~2초) 선택 영역 다시 OCR → 현재 경험치 갱신
- [x] 경과 시간 표시 (타이머)
- [x] 획득 경험치 표시 (현재 경험치 − 시작 경험치)
- [x] 시간당 경험치(EXP/h) 계산 후 표시

---

## 완료 조건

- 화면 공유 → 영역 지정 → OK → START 까지 한 흐름으로 동작
- START 후 실시간으로 경과 시간, 획득 EXP, EXP/h 가 갱신됨
